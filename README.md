# Heart_Disease_Prediction-Analysis

## Project Objective
- Analyze factors influencing cardiovascular disease
- Build a predictive Logistic Regression model
- Create an interactive Tableau dashboard for patient risk insights

## Dataset
- Source: cardio_train.xlsx
- Cleaned and pre-processed for analysis

## Python Model
- Features: Age, Gender, Cholesterol, Blood Pressure, Exercise, etc.
- Target: Disease Status (0 = No Disease, 1 = Disease)
- Model: Logistic Regression
- Accuracy: 86.8%

## Tableau Dashboard
- Charts included:
  1. Age vs Heart Disease (Bar Chart)
  2. Gender vs Heart Disease (Bar Chart)
  3. Cholesterol vs Blood Pressure (Scatter Plot)
  4. Exercise-induced Angina vs Heart Disease (Bar Chart)
- Filters: Age, Gender, Cholesterol, Blood Pressure, Angina Status
- Color-coded by Disease Status (Red = Disease, Green = No Disease)

## Insights
- Older age, high cholesterol, high BP, male gender, and exercise-induced angina increase heart disease risk.
- Dashboard enables **interactive patient risk analysis**.

## Tools & Technologies
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)
- Tableau
- Logistic Regression
